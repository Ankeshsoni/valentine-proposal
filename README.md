# Valentine Proposal Website

A simple and romantic website to propose to your Valentine.

## Features
- Romantic landing page with "Yes" and "No" buttons.
- "No" button evades the cursor.
- Photo upload functionality.
- Floating hearts animation.
- Mobile responsive.

## Running Locally

1. Install dependencies:
   ```bash
   poetry install
   ```

2. Run the server:
   ```bash
   poetry run uvicorn main:app --reload
   ```

3. Open `http://127.0.0.1:8000`.
